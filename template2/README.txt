Claremont Homeless Advocacy Program

To run the website, you can open any of the .html files to view that webpage. The links do not work when running locally, nor do the arrows on the carousel of the homepage. The website is currently hosted at chap2new.samanthastilson.com, but will move to another permanent address in the next couple of months.

[Start Bootstrap](http://startbootstrap.com/) - [Agency](http://startbootstrap.com/template-overviews/agency/)
